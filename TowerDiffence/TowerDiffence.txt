// ストーリー
・流しそうめん
　そうめん食べ放題！
　敵のHP：そうめんの濃度/色んなものを流す
　速さ：流れる道の傾き

// 概要
　一般的な通路侵入型タワーディフェンス。
　敵が目的地に到達しないようにユニットを設置する。

・流しそうめん
　(目的)
　そうめんが最後まで流れて地面に落ちてしまってはもったいない！
　多くの人を配置して、そうめんを食べる。

　(難易度)
　そうめんの量は難易度によって異なる。

　(敵HP)
　そうめんは何本かかたまって流れてくる。
　残り本数によって画像を変えるorバー表示
　(敵速度)
　そうめんの速さはそうめんの重さによって異なる。
　(敵攻撃力)
　そうめんによってグラム数が異なり、満腹度の増え方が違う。

　(ユニット設置資金)
　ウェイブ終了ごとに評判が上がる。
　評判がいいと人を多く呼ぶことが出来る。
　(ユニット強化資金)
　ユニットがそうめんを食べるごとに資金が増える。
　資金を使ってユニットに薬味を与えることが出来る。
　(ユニット攻撃力)
　一度にすくえる本数は人よって違う。
　(ユニット速さ)
　食べる速さも人によって違う。
　(ユニット消耗)
　そうめんを食べると満腹度が上がる。
　人によって満腹度の上限が決まっており、それを超えるとその人はいなくなる。
　時間経過で満腹度は少しずつ減っていく。

// ゲームシステム

・通路侵入型
　敵は、あらかじめ用意された通路を通って侵入しようとする。

　・流しそうめん
　　ステージごとにそうめんを流す道が決まっている。

・ノンストップ
　ウェイブ間で時間が止まることはない。
　ユニットの設置は素早く行わなければならない。

　・流しそうめん
　　そうめんを流す人の準備ができ次第流れてくる。

// ユニット
　ユニットを通路以外の場所に設置して、敵の侵入を防ぐ。
　ユニットごとに攻撃範囲、攻撃力、設置コストなどが異なる。

　・流しそうめん
　　人を呼んでそうめんを食べてもらう。
　　(設置資金)
　　評判が高いほど多くの人を呼ぶことが出来る。
　　ウェイブ終了に評判が上がる。
　　(強化資金)
　　人がそうめんを食べると資金が集まる。
　　集まった資金を使って薬味を与えることが出来る。
　　(攻撃力)
　　人によって一度にすくえる本数は異なる。
　　(速さ)
　　人によって食べる速さは異なる。
　　(消耗)
　　人がそうめんを食べると満腹度が上がる。
　　満腹度は自動的に少しずつ減っていく。
　　人ごとに満腹度の上限が決まっており、上限を超えるとその人はいなくなる。
　　(攻撃範囲)
　　人によって手の長さは異なる。

// ユニット特性
　攻撃範囲/攻撃力/設置コスト

// 敵
　敵はある1箇所から、一定の道を通って侵入してくる。
　倒されずに目的地に到達すると、自分のライフが減る。

　・流しそうめん
　　そうめんが流れてくる。
　　(HP)
　　そうめんは何本かまとまって流れてくる。
　　(速度)
　　そうめんの種類によって重さなどが異なる∴速さが異なる。
　　(攻撃力)
　　人が食べるそうめんにそうめんのグラム数によって満腹度の上がり方が異なる。

// 敵特性
　重さ(色)
